// Pengalaman

Apa yang Saya Kerjakan?

AWS Cloud Engineer
// Cloud Computing
  • Membangun dan mengelola website pada layanan Amazon EC2.
  • Melakukan konfigurasi Nginx sebagai web server.
  • Mengimplementasikan DNS dan domain untuk publikasi website.
  • Mengonfigurasi Load Balancer untuk meningkatkan ketersediaan layanan.
  • Mengelola keamanan server menggunakan Security Group dan IAM.
  • Deployment website berbasis cloud menggunakan EC2 terintegrasi dengan RDS dan S3.
  • Mengimplementasikan arsitektur AWS private-only dengan akses EC2 melalui EC2 Instance Connect Endpoint (EICE) dan integrasi layanan menggunakan VPC Endpoint dan terhubung ke S3 melalui S3 gateway VPC endpoint.
System Administrator
// Linux Server
  • Mengelola dan memelihara server Linux untuk memastikan ketersediaan layanan.
  • Melakukan instalasi, konfigurasi, dan pembaruan sistem operasi serta aplikasi server.
  • Konfigurasi network di dalam Linux.
  • Konfigurasi SSH, SFTP, SAMBA, Web Server, Mail Server, Automation Ansible, DHCP dan DNS Server.
Network Engineer
// Networking
  • Konfigurasi routing statis dan dinamis (OSPF, BGP) menggunakan MikroTik RouterOS.
  • Implementasi VLAN, trunking, dan inter-VLAN routing pada jaringan enterprise.
  • Konfigurasi Firewall, NAT, dan manajemen bandwidth menggunakan MikroTik.
  • Membangun topologi jaringan WAN dan LAN menggunakan Cisco Packet Tracer.
  • Konfigurasi VPN (PPTP, L2TP, IPSec) untuk koneksi remote yang aman.
Cyber Security
// Security
  • Memahami konsep dasar keamanan siber: CIA Triad, ancaman, dan vulnerability.
  • Praktik pengujian keamanan jaringan menggunakan tools seperti Nmap dan Wireshark.
  • Implementasi hardening server Linux: disable unnecessary services, SSH key auth, UFW firewall.
  • Pemegang sertifikasi Cisco Cybersecurity Essentials dan Cybersecurity Dasar.
  • Memahami teknik Social Engineering, Phishing, dan cara mitigasinya.
